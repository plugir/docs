Notification

The Notification Alert tab provides a customizable notification system to alert users when they attempt a restricted action on your site. This feature ensures that users are aware of the actions that are not permitted, enhancing the overall security and user experience.

Features​

  1. Enable/Disable Notification:

    • Toggle the notification feature on or off based on your needs. This allows you to control when the notifications should be shown to the users.

  2. Custom Notification Message:

    • Set a personalized message that will be displayed to users when they attempt a restricted action. This message can be tailored to inform users why the action is not allowed or to provide additional guidance.

How It Works​

When a user attempts an action that is restricted by the ContentGuard plugin, a notification message will be displayed. This can be useful for informing users about the site's content protection policies and preventing confusion or frustration.

Using the Notification Alert​

  1. Enable Notification:

    • To enable the notification feature, check the "Enable Notification" option. This will activate the notification system on your site.

  2. Set Custom Notification Message:

    • In the provided text area, enter the message you want to display to users. For example, you could write, "This action is not allowed on our site to protect our content."

Example​

  • Default Message:
    • "This action is not allowed!"
  • Custom Message:
    • "To maintain the security and integrity of our content, this action is restricted. Please contact support if you need further assistance."

Practical Applications​

  • Inform Users:

    • Clearly communicate to users why certain actions are restricted, enhancing their understanding and compliance.

  • Prevent Unauthorized Actions:

    • Deter users from attempting to bypass security measures by providing immediate feedback when a restricted action is attempted.

  • Enhance User Experience:

    • Use friendly and informative messages to guide users, ensuring they have a positive experience while adhering to your site's security policies.

The Notification Alert feature is a powerful tool for maintaining site security while keeping users informed. By providing real-time feedback, you can prevent unauthorized actions and ensure that users understand and respect your content protection measures.
